In order to verify that `(car (cons x y))` yields `x` for any objects `x` and~`y`, we use the substitution model:
\begtt\scm
(car (cons x y))
(car (lambda (m) (m x y)))
((lambda (m) (m x y)) (lambda (p q) p))
((lambda (p q) p) x y)
x
\endtt
The corresponding definition for `cdr` is:
\begtt\scm
(define (cdr z)
  (z (lambda (p q) q)))
\endtt
